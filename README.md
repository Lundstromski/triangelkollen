# Triangelkollen

A modern dashboard for the **Jämtlandstriangeln** hiking trail.

The goal is to provide hikers with a quick overview of the most important information in a single, easy-to-read interface.

## Features

* 🔺 Interactive trail overview
* 🌤 4-day weather forecast for each station
* 🟢 Open/closed station status
* 🍽 Station facilities (restaurant, shop, accommodation)
* 🥾 Stage information (distance, estimated hiking time, notes)
* 👥 Weekly visitor level
* 📏 Total trail distance

## Tech Stack

* React
* Vite
* SCSS
* Font Awesome
* SMHI Weather API
